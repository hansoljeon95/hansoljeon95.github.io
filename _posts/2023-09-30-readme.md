# 안내하는 글
## 로마자 표기법
Yale Romanization을 사용하지만, 아래와 같은 차이점이 있습니다.
- 'ㅸ'와 '음가 있는 ㅇ'은 각각 'v, g'로 씁니다.
- 'ㅗ'는 항상 'wo'로 씁니다.
- 율동 규칙에 의해 고저가 결정되는 음절에는 초분절음소를 명기하지 않습니다.
- 첫 거성이 나타나기 이전의 평성은 모두 표기하지 않습니다.
- 실제 용례의 초분절음소는 필요한 부분만 표기합니다.
